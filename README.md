# OPNsense 折腾手记

## 介绍

OPNsense 防火墙的安装以及折腾手记。  

- OPNsense 适用版本：24.7 - "Thriving Tiger"  

- 演示机：
    - CPU：N6005
    - 内存：16GB DDR4
    - 网卡：I226-V
    - 硬盘：256GB NVMe

- 外部网络：
    - 光猫：桥接模式
    - OPNsense：`PPPoE`

- 内部网络：
    - IPv4 网络
        - IP 地址：`172.16.1.1`
        - 子网掩码：`255.255.255.0` ( 即 `/24` )
        - 网关：`172.16.1.1`
    - IPv6 网络
        - 分配方式：`SLAAC`
        - GUA 前缀：`Prefix Delegation`
